TrackSuite.work — apt repository

Install the desktop app on Debian / Ubuntu and get updates with apt upgrade.

1. Add the signing key

curl -fsSL https://julianquandt.github.io/TrackSuite.work/apt/tracksuite-work.asc \
  | sudo gpg --dearmor -o /usr/share/keyrings/tracksuite-work.gpg

2. Add the repository

echo "deb [signed-by=/usr/share/keyrings/tracksuite-work.gpg] https://julianquandt.github.io/TrackSuite.work/apt stable main" \
  | sudo tee /etc/apt/sources.list.d/tracksuite-work.list

3. Install

sudo apt update
sudo apt install track-suite-work

Future updates arrive with your normal sudo apt update && sudo apt upgrade. Prefer not to add a repo? Grab the .deb, AppImage, .rpm or Flatpak from the latest release.
